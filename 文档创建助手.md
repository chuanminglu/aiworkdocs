# 文档创建助手

这是一个简单的PowerShell脚本，帮助您快速创建基于模板的文档。

## 使用方法

1. 在PowerShell中运行脚本
2. 选择要创建的文档类型
3. 填写相关信息
4. 脚本会自动创建文档并放置在正确的位置

## 支持的文档类型

- 日报 (daily-report)
- 技术研究笔记 (tech-research)
- 项目复盘 (project-review)
- 周报 (weekly-report)
- 会议记录 (meeting-notes)

## 脚本功能

- 自动生成文件名（包含时间戳）
- 根据内容自动选择存放位置
- 支持交叉索引
- 自动填充基本信息

## 示例

```powershell
# 创建今日的DevOps质量工程研究笔记
./new-doc.ps1 -Type "tech-research" -Topic "DevOps质量工程" -Domain "DevOps"

# 创建日报
./new-doc.ps1 -Type "daily-report"

# 创建项目复盘
./new-doc.ps1 -Type "project-review" -ProjectName "Alpha项目"
```

## 自定义模板

您可以在 `模板库/` 目录下添加自己的模板文件，格式为 `{模板名称}模板.md`。

模板中可以使用以下占位符：
- `{date}` - 当前日期
- `{timestamp}` - 当前时间戳
- `{year}` - 当前年份
- `{month}` - 当前月份
- `{filename}` - 生成的文件名
- 以及其他自定义占位符

## 高级功能

- 支持标签自动建议
- 支持关联文档自动发现
- 支持内容智能分类
- 支持重复内容检测
